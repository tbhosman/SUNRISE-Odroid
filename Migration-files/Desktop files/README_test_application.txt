Running the application:

1)
open terminal window: ctrl+alt+T

2)
Go to the right directory by typing: 
cd workspace/BicycleShed/src/ 

3)
Start application:
./BicycleShed

_______________________________________________________

Editing/compiling application
1)
open terminal window: ctrl+alt+T

2)
Go to the right directory by typing: 
cd workspace/BicycleShed/src/ 

3)
Edit the right file with a text editor

4)
Compile project by typing:
make

_______________________________________________________

make settings are in Makefile in the src directory
